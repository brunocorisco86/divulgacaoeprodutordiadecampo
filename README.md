o repositorio destina-se a criar videos de divulgação para um estande da C.Vale no Dia de Campo C.Vale 2025, ser realizado a partir do dia 02/12/2025
Local: Aviario Modelo, galpao em formato de aviario de frango de corte, destinado a divulgacao de fornecedores de avicultura, realizacao de palestras e divulgacao de iniciativas da cooperativa.
Minha empresa: C.Vale Cooperativa Agroindustrial 

## Premissas
- A C.Vale adquiriu 1000 licenças para o aplicativo eProdutor, nos módulos avicultura e financeiro. 
- Contaremos com uma televisao rodando videos sobre a parceria e chamando os produtores para se cadastrarem. Teremos eu e mais dois funcionarios do CS (Customer Sucess e CX para nos ajudar a divultar 
- Alem do video quero deixar uma tela rodando slides chamando o produtor para se cadastrar

## Prompt
- Pegue as paletas de cores dos logos da C.Vale e eProdutor
- Crie um todo para criar os slides com prompts de geracao de slides com chamadas para o cadastro, leia o conteudo de /images para que possa ser incorporado na etapa a seguir de criação de slides
- Leia o knowledge do projeto em /knowledge para ter mais contexto do projeto
- Crie slides chamando o produtor para de cadastrar
- Crie 5 videos curtos para a divulgacao
